Including makef.io

For the initiated, who already use Makefiles, you can include makef.ios very simply by including the top of your Makefile.

$(shell test -f Makef.io || curl -o Makef.io makef.io/python) include Makef.io

`test -f Makef.io` will check to see if a `Makef.io` exists, the `curl` will download the file if it does not. Notice the `/python`. This will download a stock Makefile for python. It includes `test`, `clean`, `compile` targets. You may mix multiple Makef.ios together by continuing the slashes. makef.io/python/travisci will include python targets and a target to build a `.travis.yml` build script for the project.
